Input Device

Devices that send data into a computer and transform data into binary numbers, like keyboards, cameras, microphones etc. [go back to contents]

Output Device

Devices that transform data in a computer into human receivable data, like speakers, screens, printers etc. [go back to contents]

Central Processing Unit

A device that processes all instructions of programs. It's like the brain of computer and control everything. [go back to contents]

Arithmetic Logic Unit

A integrated circuit within CPU that performs arithmetic logic operation. [go back to contents]

Control Unit

A circuit that directs operations within a CPU. [go back to contents]

Random Access Memory

A high-speed component in a device that stores information temporarily. [go back to contents]

Optical Memory

Optical memory is an electronic storage medium that uses a laser beam to store and retrieve the data. [go back to contents]

Magnetic Memory

It's a device that store memory through the alignment of small grains of magnetic material, like a megnetic disk. [go back to contents]

Flash Memory

Flash memory is an electronic non-volatile computer memory storage medium that can be electrically erased and reprogrammed, like a USB. [go back to contents]

Logic Gate

It's a tiny device that processes one or more binary numbers into a single binary number. [go back to contents]
