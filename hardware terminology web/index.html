Input Device

Devices that send data into a computer, like a keyboard, camera, microphone etc.

Output Device

Devices that tranform data in a computer into human receivable date.

Central Processing Unit

A device that processes all instruction that a computer received.

Arithmetic Logic Unit

A integrated circuit within CPU that performs arithmetc logic operation.

Control Unit

A circuity that directs operations within a CPU.

Ramdom Access Memory

A highspeed compunent in a device that store information teporarily.

Optical Memory

Optical memory is an electronic storage medium that uses a laser beam to store and retrieve the data.

Magnetic Memory

It's a device that store memory through the aligment of small grains of magnetic material.

Flash Memory

Flash memory is a long-life and non-volatile storage chip.

Logic Gate

It's a device that process one or more binary numbers into a single binary number.
